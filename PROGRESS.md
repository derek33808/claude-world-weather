## 如何运行

```bash
cd projects/claude-world-weather
npm run dev
```

## 功能使用

- **搜索城市**: 顶部搜索框输入城市名
- **温度单位**: 点击 °C / °F 按钮切换
- **查看天气**: 点击地球或城市标记
- **旋转地球**: 鼠标拖拽
- **缩放**: 滚轮
